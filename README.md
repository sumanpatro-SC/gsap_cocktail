# GSAP Cocktail

A playful interactive website showcasing cocktail recipes and animations built with React, GSAP, and Vite.

## Prerequisites

- Node.js (v16 or newer) and npm installed on your system.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm run dev
   ```
3. **Open in browser:**
   Visit `http://localhost:****` (or the URL shown in the terminal) to view the app.

## Project Structure

- `src/` contains React components and styles
- `public/` holds static assets (images, fonts, videos)
- `gsap_cocktails/` is the root with configuration files and entry points

## Build

To create a production build:

```bash
npm run build
```
